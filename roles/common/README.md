# Common Role

## Role Variables

| Name                 | Description    | Default     | Required |
| -------------------- | -------------- | ----------- | :------: |
| security soft limits | soft nproc     | `128000`    |    no    |
| security hard limits | hard nproc     | `128000`    |    no    |
| ntp_server           | server ntp     | `ntp.ix.ru` |    no    |
| ntp_type             | server or pool | `server`    |    no    |
